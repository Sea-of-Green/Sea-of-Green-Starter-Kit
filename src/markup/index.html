Sea of Green Starter Kit

This project is a set of files with which to start your Sea of Green web projects. It includes the baseline Sass files to start a project as well as favicons, logos, and this very homepage (which is a great start to your HTML as well).

Logos & Favicons

In the /assets directory, you will find the logos for Sea of Green, Hands-on Hydro, and the Green Member program. All logos & lockups are available as .pngs with transparency as well as vector formats (.ai & .svg). There is also an .svg file that contains all Sea of Green & Green Member logos as SVG symbols.

The /dist/images directory contains an ungodly amount of favicons. This is to support a wide array of browsers & mobile devices. The code for all of the favicons is included in the head this HTML file.

Stylesheets

Sea of Green uses Sass with Bourbon & Neat for stylesheets. We use a 6-1 architecture based on Sass Guidelines with a BEM naming scheme. Gulp handles our compilation, compression & concatenation needs (there's a default gulpfile.js located in the root folder).

Included Components:

  • Typography
  • Buttons
  • Input Bar
  • Footer
